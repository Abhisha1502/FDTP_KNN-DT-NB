# Food Delivery Time Prediction

This project predicts whether a food delivery will be **Fast** or **Delayed**
using Machine Learning classification techniques.

## Models Used
- Naive Bayes
- K-Nearest Neighbors (KNN)
- Decision Tree

## Features
- Distance
- Weather Conditions
- Traffic Conditions
- Order Priority
- Delivery Person Experience
- Vehicle Type
- Ratings

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Jupyter Notebook
- Matplotlib, Seaborn

## Project Structure
- `data/` – Dataset
- `src/` – Python source code
- `notebooks/` – Jupyter notebook
- `results/` – Confusion matrices & ROC curves
- `Final_Report.docx` – Project report
- `Food-Delivery-Time-Prediction.pptx` – Presentation
